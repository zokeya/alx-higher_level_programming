The ``matrix_mul`` module
==========================

Using ``matrix_mul``
---------------------

First import ``matrix_mul`` from the ``matrix_mul`` module:

	>>> matrix_mul = __import__('100-matrix_mul').matrix_mul

Now use it:

	>>> matrix_mul([[1, 2], [3, 4]], [[1, 2], [3, 4]])
	[[7, 10], [15, 22]]

	>>> matrix_mul([[1, 2]], [[3, 4], [5, 6]])
	[[13, 16]]
